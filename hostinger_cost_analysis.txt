สร้างเมื่อ: กุมภาพันธ์ 2026
โปรเจค: Thai Handwriting OCR (TrOCR + Flask)

--------------------------------------------------------------------------------
ความต้องการของระบบ
--------------------------------------------------------------------------------

- Model Size: ~1GB (best_model.pt)
- Framework: PyTorch + Transformers + Flask
- RAM ที่ต้องการ: อย่างน้อย 4-6GB (สำหรับโหลด model และ inference)
- Storage: ~2-3GB (model + dependencies)
- CPU: ควรมีอย่างน้อย 2 cores

สรุปและคำแนะนำ
--------------------------------------------------------------------------------

1. ถ้าต้องการประหยัดที่สุด:
   - ใช้ Hugging Face Spaces (ฟรี 100%)
   - แค่ต้องแก้ปัญหา version compatibility

2. ถ้าต้องการใช้ Hostinger:
   - แผน VPS KVM 2 (459 บาท/เดือน หรือ ~5,500 บาท/ปี)
   - ต้อง setup server เอง (ติดตั้ง Python, dependencies, etc.)
   - ต้องดูแล server maintenance เอง

3. ถ้าต้องการ Flask โดยไม่ใช้ Gradio และไม่อยากเสียเงินเยอะ:
   - Google Cloud Run (Free tier + pay per use)
   - แต่ต้องใช้ Docker และ setup ค่อนข้างซับซ้อน

--------------------------------------------------------------------------------
ขั้นตอนการ Deploy บน Hostinger VPS (ถ้าเลือกใช้)
--------------------------------------------------------------------------------

1. สมัคร Hostinger VPS KVM 2
2. เลือก Ubuntu 22.04 LTS
3. SSH เข้า server
4. ติดตั้ง Python 3.10+, pip, venv
5. Clone/Upload โปรเจค
6. ติดตั้ง dependencies (pip install -r requirements.txt)
7. ติดตั้ง Gunicorn สำหรับ production
8. ตั้งค่า Nginx เป็น reverse proxy
9. ตั้งค่า systemd service สำหรับ auto-start
10. ตั้งค่า SSL ด้วย Let's Encrypt

หมายเหตุ: ขั้นตอนเหล่านี้ต้องมีความรู้ Linux server administration
